# 버스 단말기 개발

## 기간

2020.07 ~ 2023.08

**C++, Qt, QML, Linux, Python**

---

## 한 줄 소개

운전자·승객 안내·결제 단말기를 개발하며, 정보 처리, 차량 디바이스 연동, 구조 개선과 양산 지원을 담당했습니다.

## 주요 기여

### 단말기 기능 개발

- 운전자·승객 안내 및 결제 단말기 UI 개발
- Qt C++ 기반 UI를 QML·C++ 기반 HMI로 전환하고, 상태·바인딩 기반 화면 갱신 구조 적용
- QML의 하드웨어 가속 렌더링 특성을 활용해 영상 재생 품질 개선
- 노선·정류장 정보 표시, 안내방송, 콘텐츠·외부 정보 연동 기능 구현
- LTE, Wi-Fi, USB, RS-232, I2C·GPIO 기반 차량 디바이스 연동
- 파일 관리, 요금, IPC 등 단말기 기능 개발

### 데이터 처리 및 개발 도구

- 노선 파일의 정류장 ID 배열과 정류장 상세 정보를 ID 기반 HashMap으로 조합해 UI용 단일 노선 데이터 생성 구조 설계
- 정류장 정보를 반복 탐색하던 조회 방식을 인덱스 조회로 전환해 데이터 조합 복잡도를 O(n²)에서 O(n)으로 개선
- Python·PySide6 기반 Windows 파일 검증 도구 개발
- 양산 시험용 테스트 프로그램 개발 및 양산 과정 지원

### 구조 및 안정성 개선

- 데이터 처리와 UI 작업을 분리해 화면 지연 문제 개선
- 명령 요청마다 독립 QThread에서 작업을 수행하고, 새 명령 처리 전 이전 작업 스레드와 객체를 정리하도록 실행 흐름 개선
- 기능을 모듈화하고 MVC를 적용해 유지보수 구조 개선
- 교통약자 정보 접근성을 위한 안내 기능 개발
